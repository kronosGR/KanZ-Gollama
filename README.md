# KandZ - Gollama

**Work under progress**

An GUI application for Ollama. The user can manage models, have multiple conversations with different model. Also to copy/paste parts of the conversations.

## Technologies

- Electron.js
- React.js
- TypeScript
- Tailwind CSS
- SQLite - to store all the conversations
